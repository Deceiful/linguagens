Javascript: O javascript é uma linguagem de alto nível responsável por dar interatividade a uma página da web, ela é interpretada, de tipagem dinâmica, orientada a objetos,e muito usada para desenvolvimento web
Java: A Java também é uma linguagem de alto nível e também é uma linguagem orientada a objetos, porém ela é fortemente tipada, ela também, diferente do javascript pode ser executada em qualquer lugar, ela pode ser tanto compilada quanto interpretada,e, diferentemente do javascript, ela é projetada pra qualquer lugar, não pra web em específico. além disso, o Java apresenta um forte enfâse em segurança.
